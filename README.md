# Lógica de Programação com Portugol

Este repositório contém meus primeiros passos na programação, focando em algoritmos e estrutura de dados básica usando Portugol Studio.

## 📌 Sobre o projeto
O objetivo destes exercícios foi consolidar os conceitos fundamentais de programação antes de migrar para linguagens de alto nível como Java.

## 🚀 Conceitos Praticados
* Variáveis e Tipos de Dados
* Estruturas Condicionais (Se/Senão)
* Estruturas de Repetição (Para/Enquanto)
* Vetores e Matrizes
